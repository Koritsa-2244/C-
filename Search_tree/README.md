Дерево поиска. Частотный словарь.

Составить программу, реализующую частотный словарь текста.
Программа должна производить следующие действия;
1. Выделять в выбранном текстовом файле отдельные слова.
2. Помещать слова в дерево поиска.
3. Выводить из дерева частотный словарь слов в алфавитном порядке.
4. Выводить частотный словарь слов, отсортированный в порядке частоты их употребления.
5. Производить поиск заданного слова.
6. Производить фильтрацию, оставляющую в словаре слова определенной длины.

Алгоритм программы:
Открываем файл (если не открыли выводим сообщение)
Создаем корень будущего дерева. Читаем слова, очищаем все слова от знаков препинаний и спец символов, сноски удаляем, первую букву делаем заглавной. Вносим в дерево по принципу: если слово раньше по алфавиту – налево, если позже – направо, если это то же слово – добавляем в частоту 1. Выводим дерево прямым обходом в словарь, таким образом получаем алфавитный словарь.
Создаем массив указателей на структуру, сортируем методом пузырька по частоте. Выводим массив с конца (сначала самые частые слова) – получаем частотный словарь
Пользователь вводит слово для поиска, программа выводит частоту с которой появляется слово, если слово не найдено, выводит соответствующее сообщение.
Пользователь вводит искомую длину, находим все слова с заданной длиной, выводим их в отдельный файл.
